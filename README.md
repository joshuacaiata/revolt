# Revolt
A custom clothing web store. This store has a custom user interface, product listings, checkout and payment form! This store is fully functional, so you can view items, add to cart, and finally, enter your own payment and shipping information. This web store is connected to the Commerce.js e-commerce platform.

## Technologies Used
This was build using JavaScript, React, Node.js, HTML, and CSS

## Product Demo
This is the front page of Revolt:
<img width="1440" alt="revoltfront" src="https://user-images.githubusercontent.com/92757478/139330328-a02b19f5-d606-4e1c-8bfb-1faea805e804.png">
The front page of the store contains all the product offerings, as well as prices, images, and a description of each item. The site is integrated with Commerce.js, so as products get added or removed from there, the store will update automatically. 

When the user clicks the shopping cart icon in the top right hand corner, it takes them to the shopping cart:
<img width="1440" alt="Screen Shot 2021-10-28 at 4 26 37 PM" src="https://user-images.githubusercontent.com/92757478/139330439-c84c0a7b-0463-41e7-985b-fe6ce8b5ad81.png">

Users can view the product offerings and add them to cart as desired. In the cart, users can edit the amount of each item they would like, although the program defaults to one. The subtotal can be found at the bottom of the page.

When a user clicks next, it simply takes them to the shipping information page:
<img width="1440" alt="Screen Shot 2021-10-28 at 4 29 16 PM" src="https://user-images.githubusercontent.com/92757478/139330851-a5b88119-4a57-4af7-8b7d-225cc194022e.png">

Once shipping information is entered and processed, the application take the user to the final step of the shopping process: payment! This is what that form looks like:
<img width="1440" alt="Screen Shot 2021-10-28 at 4 33 30 PM" src="https://user-images.githubusercontent.com/92757478/139331381-73605316-4e42-4ee3-b968-28a89f20ef05.png">
This payment form is integrated with Stripe, so it's fully functional and can process card payments.
